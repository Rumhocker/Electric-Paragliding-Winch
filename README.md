# Open Source Paragliding Winch

Zur Kommunikation zwischen der Winde und dem Sender wird ein ESP32 Lora Funkmodul ~~V3.2~~ V4 von Heltec verwendet.

![HeltecLoRaV3.2](https://github.com/Rumhocker/Electric-Paragliding-Winch/blob/main/Bilder/pinout%20V3.2.png)

Als Motor nutze ich den  QS205 v3 50H - Torque5T. 

![QS Motor](https://github.com/Rumhocker/Electric-Paragliding-Winch/blob/main/Bilder/QS205 v3 50H.jpg)

Dazu einen Vesc Controller von Makerbase 100-300 

![Vesc](https://github.com/Rumhocker/Electric-Paragliding-Winch/blob/main/Bilder/Makerbase Vesc 100300.png)
